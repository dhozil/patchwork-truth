# Patchwork Truth - Game Design Draft

## Core Loop

1. Room creation and player join.
2. Fragment distribution.
3. Negotiation phase.
4. Final narrative draft.
5. Intelligent Contract evaluation.
6. Optimistic Democracy challenge window.
7. Leaderboard and XP distribution.

## Constraints

- Non-trivia
- Unique social reasoning gameplay
- Match duration between 5 and 15 minutes
- Weekly replayability through rotating seeds

## MVP Notes

- Keep one room mode first.
- Start with one scenario seed per match.
- Add challenge stake as a fixed XP amount.
